# Bit Operation Hover Examples

이 폴더에는 **Bit Operation Hover** 기능을 테스트할 수 있는 예제 파일들이 있습니다.

## 사용 방법

### 1. 기능 활성화

VSCode 설정에서 다음 항목을 활성화하세요:

- `taskhub.experimental.bitOperationHover.enabled` = `true`

### 2. 예제 파일 열기

- **bit_operations_example.h** - 헤더 파일 형식의 다양한 비트 연산 패턴
- **bit_operations_example.cpp** - 실제 값이 있는 실용적인 예제

### 3. 사용법

지원되는 비트 연산자에 마우스를 올리면 계산 결과가 표시됩니다.

## 지원되는 패턴

#### ✅ 가능한 경우

**1. 변수와 리터럴의 비트 연산**
- `value |= 0x80` - 할당 연산자 (|=, &=, ^=, <<=, >>=)
- `value & 0xFF` - 비트 연산자 (&, |, ^, <<, >>)
- `~value` - NOT 연산
- ⚠️ **중요**: 오른쪽 피연산자는 **반드시 숫자 리터럴**이어야 합니다

**2. 상수 표현식 (NEW!)**
- `1U << 5` - 상수 시프트 연산 (항상 결과 표시)
- `0xFF & 0x0F` - 상수 AND 연산
- `0x80 | 0x40` - 상수 OR 연산
- `(1UL << 12)` - 괄호로 묶인 상수 표현식
- `256 >> 4` - 상수 right shift 연산

#### ❌ 지원되지 않는 경우

- `a ^ b` - 변수와 **변수** 간의 비트 연산
- `value &= func()` - 함수 호출 결과와의 연산
- `value &= (1 << bit)` - 복잡한 표현식 (단, 상수 표현식 `1 << bit` 부분은 hover 가능)

## Hover 정보 예시

### 변수 비트 연산

비트 연산자에 마우스를 올리면 다음과 같은 정보가 표시됩니다:

```
### Bit Operation Result

Expression: value |= 0x80

|        | Hex          | Dec | Bin                                  |
|--------|--------------|-----|--------------------------------------|
| Before | 0x0000000F   | 15  | 0b00000000000000000000000000001111 |
| After  | 0x0000008F   | 143 | 0b00000000000000000000000010001111 |
```

### 상수 표현식

상수 표현식에 마우스를 올리면 계산 결과가 표시됩니다:

```
### Constant Expression Result

Expression: 1U << 5

|        | Hex          | Dec | Bin                                  |
|--------|--------------|-----|--------------------------------------|
| Left   | 0x00000001   | 1   | 0b00000000000000000000000000000001 |
| Result | 0x00000020   | 32  | 0b00000000000000000000000000100000 |
```

## 제한사항 및 동작 방식

### 변수 비트 연산의 경우

**Before 값 표시 조건:**
- 변수의 값을 **정적으로 분석 가능**할 때만 Before 값이 표시됩니다
- LSP를 통해 변수 정의를 찾고, 초기값을 추적합니다

**✅ Before 값을 알 수 있는 경우:**
```cpp
uint8_t value = 0x0F;
value |= 0x80;  // Hover: Before=0x0F, After=0x8F
```

**❌ Before 값을 알 수 없는 경우:**
```cpp
void process(uint8_t value) {
    value |= 0x80;  // 파라미터 값을 모름 - After만 표시
}

uint8_t data = sensor_read();
data &= 0xFF;  // 함수 반환값 - After만 표시
```

### 상수 표현식의 경우

- **항상 결과를 표시**합니다 (정적 계산 가능)
- 변수가 아니므로 Before/After 대신 Left/Result로 표시됩니다

```cpp
#define BIT5 (1U << 5)  // Hover: Left=1, Result=0x20
```

## 팁

- **#define 매크로**: `#define BIT5 (1U << 5)` 같은 매크로 정의에서 `<<` 연산자에 hover하면 결과를 확인할 수 있습니다
- **정수 접미사**: U, L, UL, ULL 등의 접미사가 붙은 상수도 정확히 처리됩니다
- **디버깅**: 변수 연산에서 Before 값이 안 보이면, 변수 정의를 찾을 수 없거나 초기값이 리터럴이 아닌 경우입니다
